How to run the DMI STUDENT RESULTS

Download the zip file
Extract the file and copy DMI STUDENT RESULTS folder
Paste inside root directory(for xampp xampp/htdocs, for wamp wamp/www, for lamp var/www/html)
Open PHPMyAdmin (http://localhost/phpmyadmin)
Create a database with name srms
Import srms file(given inside the zip package in SQL file folder)
Run the script http://localhost/DMI STUDENT RESULTS
